## 1. 標題有層級安排
```
# 第一層級標題
## 第二層級標題
### 第三層級標題
#### 第四層級標題
##### 第五層級標題
###### 第六層級標題 ```

## 2. 加粗文字[雙星框]/斜體文字文字[單星框]
``` 
輸入 **後貼入要加粗的文字，在結尾處也插入 **
輸入 *後貼入要斜體的文字，在結尾處也插入*
 ```

## 3. 引用 [> ]
``` 
> 引用內容
 ```

## 4. 嵌入程式碼 [三`框]

```
輸入  [三`框] 後換行貼入程式碼，在結尾處也換行插入  [三`框]
在開頭的  [三`框] 加上程式語言名稱跟等號，可以顯示語法標示顏色跟行號
 [三`框] ```
